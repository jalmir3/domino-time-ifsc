Como Configurar a Partida
Passo a Passo:
  1. Aguardar jogadores: Espere até ter no mínimo 2 jogadores na sala
  2. Escolher modo: Selecione Individual ou Duplas
  3. Para Duplas: Selecione 2 jogadores para cada time
  4. Iniciar partida: Clique no botão quando tudo estiver pronto
Modos de Jogo:
  • Individual: Cada jogador por si mesmo
  • Duplas: 2 vs 2 - Trabalho em equipe
Dicas Importantes:
  • Certifique-se de que todos os jogadores estão presentes
  • Em duplas, combine estratégias com seu parceiro
  • Você pode atualizar a página para ver novos jogadores
  • O botão só ficará ativo quando houver no mínimo 2 jogadores
